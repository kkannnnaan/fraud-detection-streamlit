\# Fraud Detection Streamlit App



A machine learning web application that predicts whether a financial transaction is fraudulent.



\## Features



\- Real-time fraud prediction

\- Streamlit web interface

\- Scikit-learn ML pipeline

\- User-friendly transaction input form



\## Tech Stack



\- Python

\- Pandas

\- Scikit-learn

\- Streamlit

\- Joblib



\## Files



\- `fraud\_detection.py` - Streamlit application

\- `fraud\_detection\_pipeline.pkl` - Trained model pipeline

\- `analysis\_model.ipynb` - Data analysis and model training notebook

\- `requirements.txt` - Project dependencies



\## How to Run



```bash

pip install -r requirements.txt

python -m streamlit run fraud\_detection.py

```
